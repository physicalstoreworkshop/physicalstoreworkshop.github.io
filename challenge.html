As part of this workshop, four challenges will be opened using the introduced real-world physical retail data collected in grocery store environments, named Grocery Vision 2023. This dataset contains individual shopping actions of anonymized customers using a standard US shopping cart. Four challenges are:

  • Track 1a: Video Temporal Action Localization (TAL);

  • Track 1b: Video Spatial Temporal Action Localization (STAL);

  • Track 2: Object Quantity Estimation;

  • Track 3: Appearance Based Verification-Image Retrieval;
